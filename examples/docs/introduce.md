## 关于

MO-UI 是一套基于 Vue.js 2.0 的组件库，主要服务于 PC 界面的中后台产品。

## 搭建开发环境

```bash
git clone git@github.com:mo-ui/mo-ui.git
cd mo-ui
npm install
npm run dev

## open localhost:8080
```
